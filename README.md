# ⚡ Lightweight Python QR Generator

A minimal, dependency-efficient Python tool to generate high-quality QR codes from any URL or text.

## 🌟 Features
- **Adjustable Error Correction:** Support for L, M, Q, and H levels.
- **Custom Styling:** Change foreground and background colors.
- **Lightweight:** Built on top of the reliable `qrcode` and `Pillow` libraries.
